# 🛒 AK Ecommerce Platform

Production-grade MERN stack ecommerce platform.

## Tech Stack
- **Frontend:** Next.js 13 + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB + Mongoose
- **State:** Redux Toolkit
- **Auth:** JWT (Access + Refresh Tokens)

## Quick Start

### Backend
```bash
cd server
npm install
cp .env.example .env
npm run dev
